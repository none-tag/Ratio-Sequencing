How the Solver Works

This tool solves the Marker Selection Problem as an Integer Linear Programme (ILP) using the Branch & Bound method.


Variables — x(i) ∈ {0,1} whether marker i is used; e(j) ≥ 0 excess pieces for size j.

Objective — Minimise Σe(j) + ε·Σx(i) (minimise excess, break ties by fewest markers).

Constraint — For each size j: Σᵢ ratio(i,j)·plies(i)·x(i) ≥ order_qty(j).


The Claude AI API acts as the solver engine, interpreting the optimisation problem and returning the cutting plan.
